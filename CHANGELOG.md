# Changelog

## [1.0.0] - 2025-01-XX

### Added
- Telegram Mini App на React/TypeScript
- Backend API на Node.js/Express
- Модуль пожертвований (единоразовые)
- Модуль подписок (садака-джария)
- Калькулятор закята
- Модуль фондов-партнёров
- Модуль целевых кампаний
- Интеграция с YooKassa и CloudPayments
- Интеграция с Elasticsearch
- Многоязычная поддержка (RU, EN, AR)
- История пожертвований и подписок
- Docker контейнеризация

### Infrastructure
- PostgreSQL база данных
- Sequelize ORM
- Winston логирование
- Docker Compose для разработки
